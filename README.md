# KaAI
KaAI 3기 Lidar팀으로 활동하면서 한 작업들을 정리해 둔 Repository입니다.

KaAI 연구실에서는 Notion이라는 workspace를 사용하여 활동 내역을 정리하였습니다.

Lidar를 처음 사용하게 된 것이라 아주 기초적이고 간단한 작업들부터 정리를 하였습니다.
